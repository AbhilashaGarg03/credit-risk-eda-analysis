# Credit Risk EDA Analysis

## Project Overview

Exploratory Data Analysis (EDA) of credit risk data to understand factors influencing credit decisions. This analysis identifies patterns, relationships, and risk indicators in customer credit profiles.

## Objective

- Analyze credit application data
- Identify key risk factors
- Visualize distributions and relationships
- Provide insights for credit decision-making

## Dataset

Credit application and approval data with customer demographics, credit history, and approval decisions.

## Analysis Performed

1. **Univariate Analysis** - Individual variable distributions
2. **Bivariate Analysis** - Relationships with credit decisions
3. **Risk Factor Identification** - Key variables influencing credit approval
4. **Statistical Testing** - Hypothesis validation
